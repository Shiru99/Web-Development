***Run the application***

```sh
$ ./main.py
```

---

***INSTALLATIONS***

- Install python3

```sh
$ sudo apt install python3.8
```

- Install pip

```sh
$ sudo apt install -y python3-pip
```

- Install eel and pyinstaller

```sh
$ sudo pip3 install eel  pyinstaller pypng
```
